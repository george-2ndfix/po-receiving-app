🖨️ How to Print

  1. Download one of the PDFs above
  2. Open it in a PDF viewer (Preview on Mac, Acrobat on Windows)
  3. Go to File → Print
  4. Select QL-810W as printer
  5. Set paper size to DK-2225 (38mm)
  6. Scale: Actual Size (NOT "Fit to Page")
  7. Hit Print!
Each PDF has 5 test labels with different customer names (short, long, medium) to verify text auto-shrinks properly. Try printing just page 1 first.
