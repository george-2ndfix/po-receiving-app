⚙️ QL-810W Print Setup — Do This Once

  1. Click Print First Label Only to test with one label
  2. A new window opens — your browser print dialog will appear
  3. Select QL-810W as your printer
  4. Paper/Label size: DK-2225 (38mm continuous)
  5. Orientation: Portrait
  6. Margins: None
  7. Scale: 100% (not "fit to page")
  8. Print ✅
💡 If scale looks wrong: Download the PNG instead, open in Photos/Preview, and print at Actual Size. The PNG is 300 DPI so it will print at exactly 38mm wide.
